# 🧪 Ejercicios — Índice

Desafíos progresivos estilo CTF orientados al contexto SOC. Cada nivel tiene pistas y soluciones.

## Niveles

| Nivel | Archivo | Temas |
|-------|---------|-------|
| 🟢 Fácil | [nivel_1_facil.md](./nivel_1_facil.md) | Base64, Hex, identificación de hashes |
| 🟡 Medio | [nivel_2_medio.md](./nivel_2_medio.md) | HMAC, AES básico, análisis de tráfico TLS |
| 🔴 Difícil | [nivel_3_dificil.md](./nivel_3_dificil.md) | RSA, forense de ransomware, análisis completo |

## Cómo usar

1. Intentar resolver sin ver las pistas
2. Si quedás bloqueado, leer la pista (spoiler oculto en markdown)
3. Comparar con la solución al final de cada archivo
4. Para los ejercicios de código: también correr los scripts en `implementaciones/`
