# ⚗️ Implementaciones — Índice

Scripts Python funcionales que demuestran cada algoritmo. Cada uno puede ejecutarse de forma independiente.

## Requisitos

```bash
pip install pycryptodome
```

## Archivos

| Script | Tema | Conceptos |
|--------|------|-----------|
| [01_codificacion.py](./01_codificacion.py) | Base64 y Hex | Encoding, detección de patrones |
| [02_hashing.py](./02_hashing.py) | MD5, SHA-1, SHA-256, HMAC | Efecto avalancha, integridad |
| [03_aes.py](./03_aes.py) | AES-CBC y AES-GCM | Cifrado simétrico, modos |
| [04_rsa.py](./04_rsa.py) | RSA — generación, cifrado, firma | Cifrado asimétrico, PKI básico |
| [05_ransomware_sim.py](./05_ransomware_sim.py) | Simulación (solo educativa) | AES+RSA como en ransomware real |

> ⚠️ Los scripts son **solo educativos**. No usar en entornos de producción o con datos reales.
