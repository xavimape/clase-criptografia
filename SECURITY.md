# Security Policy

## Objetivo del proyecto

Este repositorio tiene fines exclusivamente educativos y está orientado a:

* Criptografía aplicada
* SOC (Security Operations Center)
* Threat Intelligence
* DFIR / Forensia Digital
* Detección y análisis de amenazas

El contenido busca enseñar técnicas defensivas, análisis de tráfico, detección y comprensión de mecanismos criptográficos utilizados en entornos reales.

---

## Uso responsable

Este proyecto NO promueve:

* actividades ilegales
* acceso no autorizado
* malware real
* despliegue ofensivo en entornos productivos

Todas las demostraciones, laboratorios y ejemplos deben utilizarse únicamente en:

* laboratorios controlados
* entornos educativos
* máquinas de prueba
* escenarios autorizados

---

## Reporte de vulnerabilidades

Si encontrás:

* vulnerabilidades
* errores de seguridad
* exposición accidental de información sensible
* problemas en ejemplos o laboratorios

por favor abrir un Issue describiendo:

* archivo afectado
* comportamiento observado
* impacto potencial
* pasos para reproducir

Evitar publicar:

* credenciales reales
* IPs sensibles
* información privada
* material malicioso operativo

---

## Contenido simulado

Los ejemplos de:

* malware
* IoCs
* fingerprints JA3
* tráfico TLS
* logs
* reglas SOC

pueden estar:

* simulados
* simplificados
* adaptados pedagógicamente

No deben considerarse inteligencia accionable real sin validación adicional.

---

## Alcance

Este proyecto es una plataforma educativa open-source y no reemplaza:

* herramientas empresariales
* auditorías profesionales
* monitoreo productivo
* validaciones de seguridad formales
